# Function: getDeploymentMnemonic()

> **getDeploymentMnemonic**(`network`): `string`

Get the deployment mnemonic for a network.

## Parameters

### network

[`NetworkType`](../type-aliases/NetworkType.md)

The target network.

## Returns

`string`

The mnemonic string.

## Throws

GeneralError if mnemonic is not found or invalid.
